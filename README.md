# Technical assessment for the position of Software Engineer - Full Stack

## Project setup

1. Run `yarn` to install dependencies
2. Create `.env` file and add:
   ```
   DATABASE_URL="postgres url here"
   ```
3. Run `yarn dev` to run the app

## Test

1. Run `yarn test`
